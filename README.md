## Preview

A highly polished, modern calculator application inspired by the Windows 11 Fluent Design UI. This project goes beyond a basic script by focusing on seamless user experience, zero-lag interactions, and visual precision.

---

## Features

- **Windows 11-Inspired UI**  
  Accurately replicates the layout, spacing, and hover effects of the native OS calculator.

- **Responsive Layout**  
  Clean, modern, and responsive design using native Tkinter frames.

- **Keyboard Support**  
  Fully mapped keyboard shortcuts including Numpad, Enter, Backspace, and Escape.

- **Localized Number Formatting**  
  Automatically formats large numbers using Indonesian formatting style:
  - Thousands separator: `.`
  - Decimal separator: `,`

- **Real-time History Display**  
  Displays previous operations and calculations above the main screen.

- **Lightweight & Fast**  
  Pure Python implementation with optimized input handling for smooth and zero-lag interaction.

---

## Installation & Usage

This project uses Python's built-in GUI library, so no external dependencies are required.

### 1. Clone this repository

```bash
git clone https://github.com/ZippyNx/FluentCalc.git
cd FluentCalc
```

### 2. Run the application

```bash
python calculator.py
```
